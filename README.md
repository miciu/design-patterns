Design Patterns
===============

This project contains a collection of design patterns in Java:

Creational Patterns:
--------------------
* Abstract Factory
* Builder
* Factory Method
* Prototype
* Singleton
* Object Pool

Structural Patterns:
--------------------
* Adapter
* Bridge
* Composite
* Decorator
* Facade
* Flyweight
* Proxy
* Filter

Behavorial Patterns:
--------------------
* Chain Of Responsibility
* Command
* Interpreter
* Iterator
* Mediator
* Memento
* Observer
* State
* Strategy
* Template Method
* Visitor
